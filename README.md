
# Mobile Sales Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard for analyzing mobile phone sales data.
The dashboard helps track sales performance, product demand, and payment trends across different cities.


## Key Metrics

* Total Sales: 44K
* Total Quantity Sold: 64
* Total Transactions: 12

## Features

* City-wise sales analysis using map visualization
* Monthly quantity trend analysis
* Mobile model performance comparison
* Customer rating distribution
* Payment method analysis
* Interactive filters for brand, model, payment method, and day

## Tools Used

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)

## Insights Generated

* Identified top-selling mobile models and high-revenue cities
* Analyzed customer purchasing patterns based on payment methods
* Tracked monthly sales trends to understand product demand

## Project Files

* `mobile_sales_dashboard.pbix` – Power BI report file
* `images/dashboard.png` – Dashboard screenshot

## How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Interact with filters and visuals to explore sales data
